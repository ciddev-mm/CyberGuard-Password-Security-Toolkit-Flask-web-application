# Week 05 – User Interface Design

## Project Title

**CyberGuard – Password Security Toolkit**

---

## User Interface Design

The user interface is designed to be simple, clean, and easy to use. Users can easily access all functions from the dashboard.

---

## Pages

The system includes the following pages:

* Login
* Register
* Dashboard
* Password Strength Analyzer
* Password Generator
* Password History
* Security Tips
* User Profile
* Activity Log

---

## Navigation

After logging in, users can navigate to different pages through the dashboard menu.

The navigation includes:

* Dashboard
* Password Analyzer
* Password Generator
* Password History
* Security Tips
* User Profile
* Logout

---

## Design Tools

The following technologies will be used for the user interface:

* HTML5
* CSS3
* Bootstrap 5
* Jinja2

---

## Design Goals

The user interface should:

* Be simple and easy to use.
* Work well on desktop and mobile devices.
* Display information clearly.
* Provide a good user experience.

---

## Week 05 Progress

During Week 05, the user interface design and page structure were prepared. The navigation menu and page layout were planned before starting the system development.
