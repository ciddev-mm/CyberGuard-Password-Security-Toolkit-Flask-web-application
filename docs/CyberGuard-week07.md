# Week 07 – Core Feature Implementation

## Project Title

**CyberGuard – Password Security Toolkit**

---

## Implementation Progress

During Week 07, the core security tools and main application logics were developed and integrated. 

The following features were successfully implemented:

* **Password Strength Analyzer**: Added client-side live character checking and server-side leak list analysis.
* **Password Generator**: Built custom generator that lets users choose length and character sets.
* **Offline Password Breach Checker**: Integrated local dictionary list with 1,000 common passwords.
* **Credentials Vault**: Configured AES cryptography (Fernet) to encrypt and decrypt saved credentials.
* **PDF Export**: Integrated ReportLab library to generate and download security scan reports.
