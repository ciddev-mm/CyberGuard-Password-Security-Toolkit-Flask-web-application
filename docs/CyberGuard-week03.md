# Week 03 – Security Analysis & Planning

## Project Title

**CyberGuard – Password Security Toolkit**

---

## Security Analysis

The project focuses on improving password security and protecting user accounts. The following security features will be included in the system:

* Password Strength Analysis
* Strong Password Generation
* Weak Password Detection
* User Authentication
* Activity Log

These features help users understand password security and reduce the risk of using weak passwords.

---

## System Planning

The system will be divided into the following modules:

* Login / Register
* Dashboard
* Password Analyzer
* Password Generator
* Password History
* User Profile
* Activity Log
* Report Export

Each module will be developed and tested step by step.

---

## Database Planning

The database will store:

* User Information
* Login Records
* Password Scan History
* Activity Logs

SQLite will be used as the database for this project.

---

## Development Plan

The project will be developed in the following order:

1. Login & Register
2. Dashboard
3. Password Strength Analyzer
4. Password Generator
5. Password History
6. User Profile
7. Activity Log
8. Export Report

---

## Week 03 Progress

During Week 03, the security features and system modules were planned. The database structure and development plan were also prepared. These will be used during the implementation stage of the project.
