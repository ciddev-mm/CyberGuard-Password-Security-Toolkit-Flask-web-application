How does it work?
In a real breach checker (e.g. HaveIBeenPwned), you never send the plain text password to the server. Instead:
  1. The password is hashed using SHA-1.
  2. Only the first 5 characters of the hash are sent to the server (using k-Anonymity).
  3. The server returns all matching hashes, and the client filters them locally.
  4. This offline app searches directly in our compiled database of common credentials to protect your network bandwidth.
